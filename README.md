# expo-cyclones-on-scratch-by-lepko25
SEE THE README.MD


Expo cyclones is a french scratch project with a MIT licence.
Press P to restart and start to begin. You will discover the fascinating world of cyclones, unveiled and launched for the first time at the Collège Saint-Exupéry in October/November 2025, accessible online. A mobile version is in development! (and an English version)
Created by the Lepko collaboration of lepine908 and nokodraj
Coded using the Scratch engine on a Windows and Linux Zorin environment. Made as part of a school project.

Use this link to access it: https://scratch.mit.edu/projects/1229635773/
Research by Dragonix, text by lepine908 and nokodraj. Presented by dragonix and nokodraj for Saint Exupéry Middle School. Some sounds are from Universal Soundbank International.
